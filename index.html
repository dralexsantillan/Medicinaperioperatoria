Sobre este proyecto de medicina perioperatoria

Esta página nace con la intención de concentrar casos clínicos de medicina perioperatoria en un formato accesible, versionable y reutilizable para docencia. GitHub funciona como repositorio central de los archivos: permite llevar control de cambios, colaborar con otros colegas y utilizar los mismos casos en talleres, simulaciones y cursos en línea.

Los escenarios están diseñados para promover el razonamiento clínico: valoración de riesgo, decisiones perioperatorias, uso adecuado de recursos, y discusión estructurada con residentes, enfermería, paramédicos y equipos quirúrgicos.

Si te interesa colaborar, adaptar estos casos a tu programa de enseñanza o desarrollar nuevos escenarios para tu hospital o curso, puedes clonar el repositorio de GitHub o contactar directamente al autor desde los enlaces que se agreguen en esta página.
